## ponzigen
Ponzigen is a generator for [Ponzu](github.com/ponzu-cms/ponzu) that implements a read-through cache for content types.

## Installing

`go get github.com/bketelsen/ponzigen`


## Usage

`ponzigen github.com/your/ponzuapp/content`
